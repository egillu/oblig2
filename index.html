Git er et versjonskontrollsytem, en type programvare som kjennetegnes ved å være distribuert- og ikke mappehierarki- basert. Hastighet ved sammenfletting var et viktig designkriterium for Git.

Github er en nettbasert vertstjeneste for versjonskontroll og brukes mest av de som skriver kode og driver med programmering.
